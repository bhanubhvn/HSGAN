# HSGAN

HSGAN - HS reconstruction from RGB images, supplementary material for IEEE Trans on NNLS submission

## 1 Pre-requisite

Note that this project is implemented using Python 3.6 and PyTorch 1.0.0.

Besides, the `h5py`, `spectral`, `opencv`, and `scikit-image` libs are used for this project.

## 2 Use the code

Change the script files in `sbatch_run.sh`, then run it:

```bash
sbatch sbatch_run.sh  or  sh script.sh
```

The details of training scripts will be released soon.
